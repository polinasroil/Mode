# Руководство по использованию мода "Буровая установка"

## Быстрый старт

### 1. Создание буровой установки
Для создания буровой установки вам понадобится:
- **4 железных слитка** (I)
- **1 алмазная кирка** (P)

Разместите материалы на верстаке в следующем порядке:
```
  I  
 IPI  
  I  
```

### 2. Установка буровой установки
1. Возьмите созданную буровую установку в руку
2. Найдите подходящее место (твердый блок)
3. Установите блок буровой установки
4. Установка автоматически начнет работать

### 3. Сбор ресурсов
- Добытые ресурсы автоматически выбрасываются рядом с установкой
- Установка работает каждые 2 секунды
- Бурение происходит вниз под установкой

## Детальное руководство

### Типы добываемых ресурсов

#### Основные руды
| Блок | Результат | Количество |
|------|-----------|------------|
| Камень | Булыжник | 1 |
| Угольная руда | Уголь | 1 |
| Железная руда | Железная руда | 1 |
| Золотая руда | Золотая руда | 1 |
| Алмазная руда | Алмаз | 1 |
| Изумрудная руда | Изумруд | 1 |

#### Специальные руды
| Блок | Результат | Количество |
|------|-----------|------------|
| Редстоун руда | Редстоун | 4 |
| Лазуритовая руда | Лазурит | 4 |

#### Другие блоки
- Все остальные блоки добываются как есть
- Бедрок не может быть добыт
- Воздух игнорируется

### Оптимизация добычи

#### Размещение установок
- Размещайте установки на расстоянии 2-3 блоков друг от друга
- Избегайте размещения над лавой или водой
- Установки работают независимо друг от друга

#### Выбор места
- **Лучшие места:** Горы, пещеры, подземные шахты
- **Плохие места:** Пляжи, океаны, пустыни
- **Идеальная глубина:** Y=10-30 (уровень руд)

#### Эффективность
- Одна установка может добыть ~30 блоков в минуту
- Для массовой добычи используйте несколько установок
- Установки работают 24/7 без перерыва

### Управление установками

#### Мониторинг
- Следите за количеством добытых ресурсов
- Проверяйте, что установка не застряла
- Убедитесь, что под установкой есть блоки для добычи

#### Обслуживание
- Установки не требуют топлива или энергии
- Автоматически останавливаются при разрушении
- Можно перемещать, разрушив и переустановив

#### Безопасность
- Установки не взрываются
- Безопасны для использования рядом с домами
- Не влияют на других игроков

## Продвинутые техники

### Автоматизация сбора
1. Постройте воронку под установкой
2. Подключите к сундуку
3. Ресурсы будут автоматически собираться

### Система сортировки
1. Используйте несколько воронок
2. Настройте фильтры по типам ресурсов
3. Создайте отдельные сундуки для каждого типа

### Массовое производство
1. Создайте "ферму" из буровых установок
2. Разместите установки в сетке 3x3
3. Используйте автоматический сбор

### Интеграция с Redstone
- Установки можно включать/выключать сигналами
- Подключите к системе автоматизации
- Создайте таймеры для контроля работы

## Решение проблем

### Установка не работает
**Проблема:** Буровая установка не добывает ресурсы
**Решение:**
1. Проверьте, что активированы оба пакета мода
2. Убедитесь, что под установкой есть блоки
3. Перезапустите мир

### Низкая производительность
**Проблема:** Игра тормозит при использовании установок
**Решение:**
1. Уменьшите количество установок
2. Разместите их дальше друг от друга
3. Проверьте настройки графики

### Ресурсы не собираются
**Проблема:** Добытые ресурсы пропадают
**Решение:**
1. Проверьте, что рядом с установкой есть место
2. Убедитесь, что инвентарь не переполнен
3. Используйте воронки для автоматического сбора

### Ошибки в консоли
**Проблема:** Появляются сообщения об ошибках
**Решение:**
1. Проверьте совместимость версии
2. Убедитесь, что все файлы мода корректны
3. Переустановите мод
