# Руководство по использованию мода "Буровая установка" v1.1.0

## Быстрый старт

### 1. Создание буровой установки
Для создания буровой установки вам понадобится:
- **4 железных слитка** (I)
- **1 алмазная кирка** (P)

Разместите материалы на верстаке в следующем порядке:
```
  I  
 IPI  
  I  
```

### 2. Установка буровой установки
1. Возьмите созданную буровую установку в руку
2. Найдите подходящее место (твердый блок)
3. Установите блок буровой установки
4. Установка автоматически начнет работать и сообщит об этом

### 3. Работа установки
- **Глубина бурения:** 100-150 блоков вниз
- **Автоматическое складывание** в ближайший сундук
- **Подробные сообщения** в чат о работе
- **Уведомления** о глубине бурения

### Система глубинного бурения

#### Глубина работы
- **Начальная глубина:** С уровня установки
- **Целевая глубина:** 100-150 блоков вниз
- **Уведомления:** При достижении 100+ блоков
- **Автоматическая остановка:** При достижении 150 блоков

#### Преимущества глубокого бурения
- **Больше руд:** Deepslate руды появляются на глубине
- **Редкие ресурсы:** Ancient Debris, алмазы, изумруды
- **Эффективность:** Автоматическое достижение лучших слоев

### Умная система складывания

#### Поиск сундука
- **Радиус поиска:** 10 блоков во всех направлениях
- **Приоритет:** Ближайший сундук
- **Автоматический поиск:** При каждой добыче

#### Логика складывания
1. **Поиск сундука** в радиусе 10 блоков
2. **Попытка складывания** в найденный сундук
3. **Резервный вариант:** Выбрасывание рядом при неудаче
4. **Уведомления** о месте складывания

#### Требования к сундуку
- **Доступность:** Сундук должен быть доступен
- **Место:** В сундуке должно быть место для предметов
- **Расположение:** В пределах 10 блоков от установки

## Продвинутые техники

### Команды управления

#### Проверка статуса
```
!drill status
!дрилл статус
```
**Что показывает:**
- Количество активных установок
- Примерную глубину каждой установки
- Общий статус работы

#### Остановка всех установок
```
!drill stop all
!дрилл стоп все
```
**Что делает:**
- Останавливает все активные установки
- Освобождает ресурсы
- Показывает количество остановленных установок

### Оптимизация добычи

#### Размещение установок
- **Расстояние:** 2-3 блока друг от друга
- **Сундуки:** Размещайте сундуки рядом с установками
- **Глубина:** Устанавливайте на высоких уровнях для максимальной глубины

#### Выбор места
- **Лучшие места:** Горы, высокие плато
- **Плохие места:** Низменности, океаны
- **Идеальная высота:** Y=100+ для максимальной глубины

#### Эффективность
- **Одна установка:** ~30 блоков в минуту
- **Глубина:** 100-150 блоков за 5-7 минут
- **Автоматизация:** Работает 24/7 до достижения глубины

### Автоматизация сбора

#### Система сундуков
1. **Постройте сундук** рядом с установкой
2. **Установка автоматически** найдет и использует его
3. **Ресурсы складываются** без вашего участия

#### Множественные установки
1. **Создайте сеть** буровых установок
2. **Разместите сундуки** между установками
3. **Максимальная добыча** с минимальными усилиями

#### Мониторинг работы
1. **Используйте команды** для проверки статуса
2. **Следите за сообщениями** в чате
3. **Контролируйте глубину** бурения

## Решение проблем

### Установка не работает
**Проблема:** Буровая установка не добывает ресурсы
**Решение:**
1. Проверьте, что активированы оба пакета мода
2. Убедитесь, что под установкой есть блоки
3. Перезапустите мир

### Ресурсы не складываются в сундук
**Проблема:** Добытые ресурсы выбрасываются рядом
**Решение:**
1. Убедитесь, что сундук находится в радиусе 10 блоков
2. Проверьте, что в сундуке есть место
3. Убедитесь, что сундук доступен

### Установка остановилась рано
**Проблема:** Бурение остановилось до достижения глубины
**Решение:**
1. Проверьте, не достигнута ли глубина 150 блоков
2. Убедитесь, что блок установки не разрушен
3. Используйте команду `!drill status` для проверки

### Низкая производительность
**Проблема:** Игра тормозит при использовании установок
**Решение:**
1. Уменьшите количество установок
2. Разместите их дальше друг от друга
3. Проверьте настройки графики
