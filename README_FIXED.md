# Minecraft Bedrock Server - Исправленная версия

Полностью рабочий сервер Minecraft Bedrock Edition, написанный с нуля на Python 3.11+.

## 🚀 Возможности

### ✅ Полная реализация RakNet
- Поддержка всех пакетов RakNet
- Правильная обработка connection handshake
- Управление соединениями и таймаутами
- Статистика сетевого трафика

### ✅ Улучшенная генерация мира
- Множественные биомы (Plains, Forest, Desert, Mountains, Swamp)
- Генерация деревьев, цветов, кактусов
- Простые структуры (дома)
- Система пещер
- Шумовая генерация рельефа

### ✅ Полноценное подключение игроков
- 12-этапная последовательность подключения
- Отправка всех необходимых пакетов
- Правильное появление в мире
- Синхронизация между игроками
- Обработка отключений

### ✅ Улучшенная система пакетов
- Полная сериализация/десериализация
- Поддержка NBT формата
- BatchPacket с LZW-сжатием
- Правильная обработка всех типов пакетов

## 📋 Требования

- Python 3.11+
- Только стандартные библиотеки Python
- Никаких дополнительных зависимостей

## 🛠️ Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd minecraft-bedrock-server
```

### 2. Запуск сервера

#### Основная версия (рекомендуется)
```bash

```

#### Исправленная версия (для тестирования)
```bash

```

#### Упрощенная версия (для отладки)


### 3. Тестирование
```bash

```

### 4. Демонстрация возможностей
```bash

```

## 🎮 Подключение из игры

### Настройка в Minecraft Bedrock:

1. **Откройте Minecraft Bedrock Edition**
2. **Перейдите в "Игры" → "Серверы"**
3. **Нажмите "Добавить сервер"**
4. **Заполните поля:**
   - **Имя сервера:** `Python Bedrock Server`
   - **IP-адрес:** `192.168.1.X` (IP вашего компьютера)
   - **Порт:** `19132`
   - **Игрок:** Любое имя

### Получение IP-адреса:

#### Windows:
```cmd
ipconfig
```

#### Linux/macOS:
```bash
ifconfig
# или
ip addr show
```

### Проверка доступности:
```bash
# Проверьте, что порт 19132 открыт
netstat -an | grep 19132
```

## 📡 Протокол подключения

### Последовательность подключения:

1. **Unconnected Ping** - Проверка доступности сервера
2. **Open Connection Request 1** - Запрос на открытие соединения
3. **Open Connection Reply 1** - Ответ сервера
4. **Open Connection Request 2** - Подтверждение соединения
5. **Open Connection Reply 2** - Финальное подтверждение
6. **Connection Request** - Запрос авторизации
7. **Connection Request Accepted** - Принятие авторизации
8. **New Incoming Connection** - Подтверждение подключения
9. **Login Success** - Успешная авторизация
10. **Resource Packs Info** - Информация о ресурс-паках
11. **Resource Pack Stack** - Стек ресурс-паков
12. **Start Game** - Начало игры
13. **Spawn Position** - Позиция возрождения
14. **Time & Weather** - Время и погода
15. **Game Rules** - Правила игры
16. **Commands Enabled** - Включение команд
17. **Player Permissions** - Права игрока
18. **Spawn Chunks** - Чанки вокруг спавна
19. **Player Spawn** - Появление игрока

## 🎯 Команды сервера

### Консольные команды:

- `stop` - Остановить сервер
- `list` - Список игроков
- `say <текст>` - Отправить сообщение всем игрокам
- `kick <игрок>` - Кикнуть игрока
- `time <day|night>` - Установить время
- `weather <clear|rain>` - Установить погоду
- `gamemode <survival|creative>` - Режим игры
- `help` - Справка по командам

### Примеры использования:
```
> list
Игроки онлайн: Player_1234567, Player_8901234

> say Добро пожаловать на сервер!
[Сервер] Добро пожаловать на сервер!

> time day
Время установлено на день

> weather clear
Погода установлена на ясно
```

## 🐛 Устранение неполадок

### Проблема: Сервер не запускается
**Решение:**
```bash
# Проверьте версию Python
python3 --version

# Проверьте права доступа к порту
sudo netstat -tulpn | grep 19132

# Запустите с правами администратора (если нужно)
sudo python3 main.py
```

### Проблема: Игра не подключается
**Решение:**
1. Проверьте IP-адрес сервера
2. Убедитесь, что порт 19132 открыт
3. Проверьте файрвол
4. Попробуйте подключиться с того же устройства

### Проблема: Ошибки в логах
**Решение:**
```bash
# Проверьте логи
tail -f server.log

# Запустите с подробным логированием
python3 main.py --debug
```
