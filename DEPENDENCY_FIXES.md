# 🔧 Исправления зависимостей мода

## ❌ Проблемы, которые были исправлены

### 1. Несуществующие зависимости в manifest.json
**Было:**
```json

  {
    "uuid": "b6a8f7b3-4c6e-4f3d-8c5a-123456789012",
    "version": [1, 0, 0]
  }
]
```

**Стало:**
```json
"dependencies": []
```

### 2. Несуществующие блоки
**Было:**
```json

```

**Стало:**
```json

```

**Было:**
```json
"textures": "textures/items/drill_rig"
```

**Стало:**
```json

```
