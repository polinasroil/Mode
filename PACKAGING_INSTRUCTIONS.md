# Инструкции по упаковке мода

## Создание .mcaddon файла

### Способ 1: Автоматическая упаковка
1. Убедитесь, что все файлы находятся в правильных папках
2. Используйте специальные инструменты для упаковки модов Minecraft PE

### Способ 2: Ручная упаковка
1. Создайте ZIP архив со следующей структурой:
   ```
   drill_rig_mod.zip
   ├── manifest.json
   ├── behavior_pack.json
   ├── resource_pack.json
   ├── scripts/
   ├── textures/
   ├── models/
   ├── animations/
   ├── blocks/
   ├── items/
   ├── recipes/
   ├── loot_tables/
   ├── trading/
   └── texts/
   ```

2. Переименуйте расширение с `.zip` на `.mcaddon`

### Способ 3: Использование командной строки
```bash
# Создание архива
zip -r drill_rig_mod.zip . -x "*.git*" "*.DS_Store*" "*.mcaddon"

# Переименование
mv drill_rig_mod.zip drill_rig_mod.mcaddon
```

## Проверка структуры
Перед упаковкой убедитесь, что:
- Все UUID уникальны
- Файлы находятся в правильных папках
- Формат файлов соответствует версии Minecraft PE
- Нет синтаксических ошибок в JSON файлах

## Тестирование
1. Установите мод в Minecraft PE
2. Проверьте крафт буровой установки
3. Убедитесь, что бурение работает корректно
4. Проверьте добычу различных ресурсов

## Распространение
- Файл .mcaddon можно загружать на различные платформы
- Убедитесь, что у вас есть права на распространение
- Добавьте информацию об авторе и лицензии
