# Инструкция по установке Ore Collector Mod для Minecraft PE

## Что такое .mcaddon файл?

`.mcaddon` файл - это специальный формат для мобильной версии Minecraft PE, который содержит в себе два пака:
- **Behavior Pack** (пак поведения) - содержит логику, рецепты, поведение блоков
- **Resource Pack** (пак ресурсов) - содержит текстуры, модели, звуки


## Шаг 1: Создание .mcaddon файла

### Вариант 1: Автоматическое создание
1. Скачайте все файлы мода
2. Создайте папку `ore_collector_mod`
3. Поместите в неё папки `behavior_pack` и `resource_pack`
4. Сожмите папку `ore_collector_mod` в ZIP архив
5. Переименуйте расширение с `.zip` на `.mcaddon`

### Вариант 2: Ручное создание
1. Создайте папку `ore_collector_mod`
2. Создайте внутри неё папки `behavior_pack` и `resource_pack`
3. Скопируйте файлы в соответствующие папки
4. Сожмите в ZIP и переименуйте в `.mcaddon`

## Шаг 2: Установка на телефон

### Android:
1. Скачайте `.mcaddon` файл на телефон
2. Откройте файл - он автоматически откроется в Minecraft PE
3. Нажмите "Импорт в мир"
4. Выберите мир или создайте новый
5. Активируйте оба пака в настройках мира

### iOS:
1. Скачайте `.mcaddon` файл на iPhone/iPad
2. Откройте файл через "Поделиться" → Minecraft
3. Следуйте инструкциям по импорту

## Шаг 3: Активация в игре

1. Откройте мир с модом
2. Перейдите в **Настройки** → **Экспериментальные функции**
3. Включите **Holiday Creator Features**
4. Перейдите в **Настройки** → **Ресурс-паки**
5. Активируйте **Ore Collector RP**
6. Перейдите в **Настройки** → **Пакеты поведения**
7. Активируйте **Ore Collector BP**

## Шаг 4: Проверка работы

1. Создайте верстак
2. Скрафтите сборщик руды: 8 алмазов + 1 сундук
3. Разместите блок в мире
4. Сборщик начнет работать автоматически каждые 10 секунд

## Возможные проблемы

### Мод не работает:
- Проверьте, что оба пака активированы
- Убедитесь, что включены экспериментальные функции
- Перезапустите игру

### Ошибка импорта:
- Проверьте целостность `.mcaddon` файла
- Убедитесь, что версия Minecraft PE 1.20.0+
- Попробуйте создать новый мир

### Блок не крафтится:
- Проверьте рецепт: 8 алмазов по краям, сундук в центре
- Убедитесь, что используете верстак

## Технические детали

- **Версия Minecraft PE**: 1.20.0+
- **Формат пакетов**: 15
- **Радиус поиска руды**: 5 блоков
- **Частота сбора**: каждые 10 секунд
- **Поддерживаемые руды**: все стандартные + deepslate варианты

## Поддержка

При возникновении проблем:
1. Проверьте версию игры
2. Убедитесь в правильности установки
3. Попробуйте переустановить мод
4. Создайте новый мир для тестирования

## Безопасность

- Мод не изменяет основные файлы игры
- Можно безопасно удалить в любой момент
- Совместим с другими модами
- Не влияет на производительность
