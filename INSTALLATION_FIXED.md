1. Откройте Minecraft PE
2. Перейдите в **Настройки** → **Глобальные ресурсы**
3. Нажмите **Импорт**
4. Выберите файл `drill_rig_mod.mcaddon`
5. Дождитесь завершения импорта

5. Сохраните настройки и запустите мир

## 🔧 Как работает исправленный мод

### Принцип работы
- Мод использует существующий **железный блок** (`minecraft:iron_block`)
- При установке железного блока он автоматически становится буровой установкой
- Бурение происходит каждые 2 секунды под установкой

### Создание буровой установки
**Рецепт:**
```
  I  
 IPI  
  I  
```
- I = Железный слиток (4 шт.)
- P = Алмазная кирка (1 шт.)

### Использование
1. Создайте железный блок по рецепту
2. Поставьте его на землю
3. Блок автоматически начнет бурение
4. Собирайте добытые ресурсы рядом
