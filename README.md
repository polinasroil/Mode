# 🪙 Telegram Бот "Орёл или решка"

Полнофункциональный игровой бот для Telegram с игрой "Орёл или решка", включающий режимы игры с ботом и дуэли между игроками.

## 🎮 Возможности

### Основные функции:
- 🎯 **Игра с ботом** - классическая игра "Орёл или решка" против ИИ
- ⚔️ **Дуэли между игроками** - создавайте дуэли и сражайтесь с друзьями
- 💰 **Система баланса** - каждый игрок начинает с 1000 монет
- 📊 **Статистика игр** - отслеживайте свои победы и поражения
- 🎨 **Красивый интерфейс** - удобные inline-клавиатуры и эмодзи

### Особенности:
- ✅ **Безопасность** - проверка баланса и валидация всех действий
- ⏰ **Таймауты** - автоматическая очистка истекших игр и дуэлей
- 🔄 **Анимации** - эффекты подбрасывания монеты
- 📱 **Inline режим** - возможность делиться дуэлями через inline-запросы
- 🗄️ **База данных** - SQLite для хранения всех данных игроков

## 🚀 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone <repository-url>
cd telegram-coin-flip-bot
```

### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 3. Настройка бота
1. Создайте бота у [@BotFather](https://t.me/BotFather) в Telegram
2. Получите токен бота
3. Скопируйте файл `.env.example` в `.env`:
```bash
cp .env.example .env
```
4. Отредактируйте `.env` и вставьте ваш токен:
```env
BOT_TOKEN=your_actual_bot_token_here
```

### 4. Запуск бота
```bash
python bot.py
```

## 🎯 Как играть

### Игра с ботом:
1. Отправьте `/start` боту
2. Выберите "🎮 Игра с ботом"
3. Выберите сумму ставки (от 1 до 1000)
4. Выберите сторону монеты (Орёл или Решка)
5. Дождитесь результата и получите выигрыш!

### Дуэли:
1. Выберите "⚔️ Создать дуэль"
2. Отправьте ссылку другу
3. Оба игрока делают ставки
4. Оба выбирают сторону монеты
5. Победитель забирает все ставки!

## 📊 Команды бота

- `/start` - начать игру
- `/help` - показать правила и помощь
- `/stats` - ваша статистика игр
- `/balance` - ваш текущий баланс

## ⚙️ Настройки

Все настройки можно изменить в файле `config.py`:

```python
# Настройки игры
MIN_BET = 1              # Минимальная ставка
MAX_BET = 1000           # Максимальная ставка
DUEL_TIMEOUT = 60        # Время ожидания дуэли (секунды)
GAME_TIMEOUT = 30        # Время ожидания в игре (секунды)
```

## 🐛 Устранение неполадок

### Частые проблемы:

1. **"Не установлен BOT_TOKEN"**
   - Проверьте файл `.env` и убедитесь, что токен указан правильно

2. **"Ошибка подключения к базе данных"**
   - Убедитесь, что у процесса есть права на запись в директорию

3. **"Бот не отвечает"**
   - Проверьте логи на наличие ошибок
   - Убедитесь, что токен бота действителен

### Логирование:
Бот ведет подробные логи. Для включения отладочного режима измените уровень логирования в `bot.py`:

```python
logging.basicConfig(level=logging.DEBUG)
```
