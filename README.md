# 🎮 Minecraft PE Server - Мобильный Сервер

## 📱 Описание проекта
Этот проект представляет собой полноценный сервер для Minecraft Pocket Edition (PE), оптимизированный для работы на мобильных устройствах. Сервер поддерживает все основные функции Minecraft и предоставляет удобный веб-интерфейс для управления.

## 🚀 Основные возможности
- ✅ Поддержка Minecraft PE 1.20+
- ✅ Веб-панель управления
- ✅ Система плагинов
- ✅ Автоматическое резервное копирование
- ✅ Система пользователей и прав
- ✅ Мониторинг производительности
- ✅ REST API для интеграции

## 🛠️ Технические требования
- **Операционная система**: Android 8.0+, iOS 12.0+
- **RAM**: Минимум 2GB, рекомендуется 4GB+
- **Хранилище**: Минимум 500MB свободного места
- **Сеть**: Стабильное интернет-соединение

## 🚀 Быстрый старт

### 1. Установка
```bash
# Клонирование репозитория
git clone https://github.com/your-username/minecraft-pe-server.git
cd minecraft-pe-server

# Установка зависимостей
./scripts/install.sh

# Запуск сервера
./scripts/start.sh
```

### 2. Настройка
1. Откройте файл `config/server.properties`
2. Настройте параметры сервера
3. Перезапустите сервер

### 3. Доступ к веб-панели
Откройте браузер и перейдите по адресу: `http://localhost:8080`

## 🔧 Конфигурация

### Основные настройки сервера
```properties
# config/server.properties
server-name=Minecraft PE Server
server-port=19132
max-players=20
gamemode=survival
difficulty=normal
spawn-protection=16
```

### Настройки веб-панели
```properties
# config/web-panel.properties
web-port=8080
admin-username=admin
admin-password=secure_password
```

## 📊 Мониторинг

Сервер предоставляет следующие метрики:
- Количество игроков онлайн
- Использование CPU и RAM
- Загрузка сети
- Статистика игр

## 🔌 Плагины

### Встроенные плагины
- **CoreProtect** - Логирование действий игроков
- **WorldEdit** - Редактирование мира
- **Essentials** - Основные команды
- **PermissionsEx** - Система прав

### Установка дополнительных плагинов
1. Скачайте .jar файл плагина
2. Поместите в папку `plugins/`
3. Перезапустите сервер

- Мониторьте использование ресурсов

## 🐛 Устранение неполадок

### Частые проблемы
1. **Сервер не запускается**
   - Проверьте порты
   - Убедитесь в достаточности RAM
   - Проверьте логи

2. **Игроки не могут подключиться**
   - Проверьте настройки сети
   - Убедитесь в правильности IP адреса
   - Проверьте файрвол
