# Мод "Буровая установка" для Minecraft PE

## Описание
Этот мод добавляет в Minecraft PE автоматическую буровую установку, которая может добывать ресурсы под собой.

## Возможности
- Автоматическая добыча ресурсов
- Поддержка всех основных руд (уголь, железо, золото, алмазы и др.)
- Эффекты частиц при бурении
- Простой крафт из доступных материалов

## Установка

### Для игроков:
1. Скачайте файл `.mcaddon`
2. Откройте Minecraft PE
3. Перейдите в настройки → Глобальные ресурсы
4. Нажмите "Импорт" и выберите скачанный файл
5. Активируйте мод в мире

### Для разработчиков:
1. Скопируйте все файлы в папку мода
2. Убедитесь, что структура папок соответствует стандарту
3. Скомпилируйте в `.mcaddon` файл

## Использование

### Крафт буровой установки:
```
  I  
 IPI  
  I  
```
Где:
- I = Железный слиток
- P = Алмазная кирка

### Установка:
1. Поместите буровую установку на землю
2. Установка автоматически начнет бурение вниз
3. Добытые ресурсы будут выброшены рядом с установкой

### Удаление:
Просто разрушьте блок буровой установки любой киркой

## Технические детали

### Поддерживаемые блоки:
- Камень → Булыжник
- Угольная руда → Уголь
- Железная руда → Железная руда
- Золотая руда → Золотая руда
- Алмазная руда → Алмаз
- Изумрудная руда → Изумруд
- Редстоун руда → Редстоун (4 шт.)
- Лазуритовая руда → Лазурит (4 шт.)

### Скорость бурения:
- Бурение происходит каждые 2 секунды
- Установка работает автоматически
- Останавливается при разрушении блока
