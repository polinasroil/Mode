## 🎯 ПОДКЛЮЧЕНИЕ ИЗ ИГРЫ

### Настройка в Minecraft Bedrock:

1. **Откройте Minecraft Bedrock Edition**
2. **Игры → Серверы → Добавить сервер**
3. **Заполните поля:**
   - **Имя сервера:** `Python Bedrock Server`
   - **IP-адрес:** `192.168.1.X` (ваш IP)
   - **Порт:** `19132`
   - **Игрок:** Любое имя

### Получение IP-адреса:
```bash
# Linux/macOS
ip addr show
# или
ifconfig

# Windows
ipconfig
```

## 🐛 УСТРАНЕНИЕ НЕПОЛАДОК

### Проблема: Сервер не запускается
```bash
# Проверьте версию Python
python3 --version

# Проверьте права доступа к порту
sudo netstat -tulpn | grep 19132

# Запустите с правами администратора
sudo python3 main.py
```

### Проблема: Игра не подключается
1. Убедитесь, что запущен `python3 main.py`
2. Проверьте IP-адрес сервера
3. Убедитесь, что порт 19132 открыт
4. Проверьте файрвол

### Проблема: Ошибки в логах
```bash
# Проверьте логи
tail -f server.log

# Запустите с подробным логированием
python3 main.py --debug
```
