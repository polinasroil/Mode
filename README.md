# Мод "Буровая установка" для Minecraft PE

## Описание
Этот мод добавляет в Minecraft PE автоматическую буровую установку, которая может добывать ресурсы на глубине 100-150 блоков, складывать их в ближайший сундук и сообщать о своей работе в чат.

## Возможности
- **Глубокое бурение** на глубину 100-150 блоков
- **Автоматическая добыча** всех типов руд
- **Умное складывание** в ближайший сундук
- **Подробные сообщения** в чат о работе
- **Поддержка всех руд** включая deepslate и ancient debris
- **Эффекты частиц** при бурении
- **Простой крафт** из доступных материалов
- **Команды управления** для мониторинга и остановки

## Установка

### Для игроков:
1. Скачайте файл `.mcaddon`
2. Откройте Minecraft PE
3. Перейдите в настройки → Глобальные ресурсы
4. Нажмите "Импорт" и выберите скачанный файл
5. Активируйте мод в мире

### Для разработчиков:
1. Скопируйте все файлы в папку мода
2. Убедитесь, что структура папок соответствует стандарту
3. Скомпилируйте в `.mcaddon` файл

## Использование

### Крафт буровой установки:
```
  I  
 IPI  
  I  
```
Где:
- I = Железный слиток (4 шт.)
- P = Алмазная кирка (1 шт.)

### Установка и работа:
1. Поместите буровую установку на землю
2. Установка автоматически начнет бурение вниз
3. **Глубина бурения:** 100-150 блоков
4. **Автоматическое складывание** в ближайший сундук
5. **Подробные сообщения** в чат о работе

### Команды управления:
- `!drill status` или `!дрилл статус` - показать статус всех установок
- `!drill stop all` или `!дрилл стоп все` - остановить все установки

## Технические детали

### Поддерживаемые блоки:
- **Обычные руды:** Камень, уголь, железо, золото, алмазы, изумруды
- **Deepslate руды:** Все руды в deepslate варианте
- **Специальные:** Редстоун (4 шт.), лазурит (4 шт.), ancient debris, медь
- **Остальные блоки:** Добываются как есть

### Глубина бурения:
- **Начало:** С уровня установки
- **Максимум:** 150 блоков вниз
- **Уведомления:** При достижении 100+ блоков
- **Автостоп:** При достижении 150 блоков

### Система складывания:
- **Поиск сундука:** В радиусе 10 блоков
- **Приоритет:** Ближайший сундук
- **Резерв:** Выбрасывание рядом при отсутствии сундука
- **Уведомления:** О месте складывания ресурсов

### Скорость бурения:
- Бурение происходит каждые 2 секунды
- Установка работает автоматически
- Останавливается при разрушении блока или достижении глубины
