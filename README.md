# Ore Collector Mod для Minecraft PE

## Описание
Этот мод добавляет в игру блок "Сборщик руды", который автоматически находит и собирает руду в радиусе 5 блоков каждые 10 секунд.

## Функции
- Автоматический поиск руды в радиусе 5 блоков
- Автоматический сбор руды каждые 10 секунд
- Поддержка всех типов руды (уголь, железо, золото, алмаз и др.)
- Замена руды на камень после сбора
- Выпадение предметов руды

## Установка
1. Скачайте все файлы мода
2. Создайте папку с названием мода в папке `behavior_packs` вашего мира
3. Скопируйте все файлы в эту папку
4. Активируйте мод в настройках мира

## Крафт
Для создания сборщика руды вам понадобится:
- 8 алмазов
- 1 сундук

Расположите их в верстаке в следующем порядке:
```
DDD
DCD
DDD
```
Где D = алмаз, C = сундук

## Использование
1. Разместите блок сборщика руды в мире
2. Сборщик автоматически начнет работать
3. Каждые 10 секунд он будет искать руду в радиусе 5 блоков
4. Найденная руда будет автоматически собрана и заменена на камень
5. Предметы руды будут выброшены рядом с блоком

## Совместимость
- Minecraft PE 1.20.0+
- Поддерживает все типы руды включая deepslate варианты

## Примечания
- Сборщик работает только в измерении Overworld
- При разрушении блока сборщика его функциональность прекращается
- Радиус поиска можно изменить в файле `scripts/main.js`

## Поддерживаемые руды
- Угольная руда
- Железная руда
- Золотая руда
- Алмазная руда
- Изумрудная руда
- Лазуритовая руда
- Редстоун руда
- Медная руда
- Все deepslate варианты руд
