## 🔍 **ОТЛАДКА ПРОБЛЕМ:**

### **Если тесты не проходят:**

#### **Проблема 1: ModuleNotFoundError**
```bash
# Решение: проверьте структуру файлов
ls -la server/
# Должно быть:
# __init__.py
# bedrock_protocol_v2.py
# detailed_logger.py
# main_fixed.py
```

#### **Проблема 2: ImportError в detailed_logger**
```bash
# Решение: проверьте Python версию
python3 --version
# Должно быть 3.8+
```

#### **Проблема 3: Ошибки в протоколе**
```bash
# Запустите тест с подробным выводом
python3 -u test_detailed_logging.py 2>&1 | tee test_output.log
```

### **Если сервер запускается, но игроки не подключаются:**

#### **Проверьте логи:**
```bash
# Последние логи
tail -f logs/detailed_*.log
tail -f logs/protocol_*.log
```

#### **Проверьте сеть:**
```bash
# Проверка порта
netstat -tuln | grep 19132
# Должно показать: UDP 0.0.0.0:19132

# Проверка файрвола
sudo ufw status
# Порт 19132 должен быть открыт
```
