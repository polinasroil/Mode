## 🔄 Создание Pull Request

### Процесс создания PR

1. **Fork репозитория** - Создайте свою копию проекта
2. **Создайте ветку** - Создайте отдельную ветку для ваших изменений
3. **Внесите изменения** - Реализуйте ваши улучшения
4. **Протестируйте** - Убедитесь, что все работает
5. **Создайте PR** - Отправьте pull request

### Названия веток

Используйте понятные названия веток:

- `fix/описание-исправления` - для исправлений ошибок
- `feature/описание-функции` - для новых функций
- `docs/описание-изменений` - для изменений в документации
- `refactor/описание-рефакторинга` - для рефакторинга

### Описание PR

```markdown
## Описание изменений
Краткое описание того, что было изменено

## Тип изменений
- [ ] Исправление ошибки
- [ ] Новая функция
- [ ] Улучшение документации
- [ ] Рефакторинг
- [ ] Тесты

## Тестирование
Опишите, как вы тестировали изменения

## Скриншоты
Если применимо, добавьте скриншоты

## Чек-лист
- [ ] Код соответствует стандартам проекта
- [ ] Добавлены тесты для новых функций
- [ ] Обновлена документация
- [ ] Все тесты проходят
```

## 🧪 Тестирование

### Запуск тестов

```bash
# Установка зависимостей для тестирования
pip install -r requirements.txt

# Запуск всех тестов
pytest

# Запуск тестов с покрытием
pytest --cov=server --cov=web_panel

# Запуск конкретного теста
pytest tests/test_server.py::test_player_join
```

### Написание тестов

- Покрывайте тестами новую функциональность
- Используйте моки для внешних зависимостей
- Тестируйте как успешные, так и неуспешные сценарии

```python
import pytest
from server.main import MinecraftPEServer

@pytest.mark.asyncio
async def test_player_join():
    """Тест подключения игрока к серверу"""
    server = MinecraftPEServer()
    
    # Тест успешного подключения
    player = await server.player_join("TestPlayer", "192.168.1.100")
    assert player.username == "TestPlayer"
    assert player.ip_address == "192.168.1.100"
    
    # Тест подключения существующего игрока
    with pytest.raises(Exception, match="Игрок уже подключен"):
        await server.player_join("TestPlayer", "192.168.1.101")
```
