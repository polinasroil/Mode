# 🎮 Minecraft PE Server - Мобильный Сервер

## 📱 Описание проекта
Этот проект представляет собой полноценный сервер для Minecraft Pocket Edition (PE), оптимизированный для работы на мобильных устройствах. Сервер поддерживает все основные функции Minecraft и предоставляет удобный веб-интерфейс для управления.

## 🚀 Основные возможности
- ✅ Поддержка Minecraft PE 1.20+
- ✅ Веб-панель управления с современным интерфейсом
- ✅ Система плагинов и модов
- ✅ Автоматическое резервное копирование
- ✅ Система пользователей и прав доступа
- ✅ Мониторинг производительности в реальном времени
- ✅ REST API для интеграции
- ✅ Асинхронная архитектура на Python
- ✅ Адаптивный веб-интерфейс
- ✅ Система логирования и мониторинга

## 🛠️ Технические требования
- **Операционная система**: Linux, macOS, Windows (WSL)
- **Python**: 3.8 или выше
- **RAM**: Минимум 2GB, рекомендуется 4GB+
- **Хранилище**: Минимум 500MB свободного места
- **Сеть**: Стабильное интернет-соединение

## 🚀 Быстрый старт

### 1. Клонирование репозитория
```bash
git clone https://github.com/your-username/minecraft-pe-server.git
cd minecraft-pe-server
```

### 2. Быстрый запуск (рекомендуется для тестирования)
```bash
python3 quick_start.py
```

### 3. Полная установка (рекомендуется для продакшена)
```bash
# Установка зависимостей
pip3 install -r requirements.txt

# Запуск скрипта установки
./scripts/install.sh

# Запуск сервера
./scripts/start.sh
```

### 4. Доступ к веб-панели
Откройте браузер и перейдите по адресу: `http://localhost:8080`
- **Логин**: `admin`
- **Пароль**: `admin123`

## 🔧 Конфигурация

### Основные настройки сервера
```properties
# config/server.properties
server-name=Minecraft PE Server
server-port=19132
max-players=20
gamemode=survival
difficulty=normal
spawn-protection=16
```

### Настройки веб-панели
```properties
# config/web-panel.properties
web-port=8080
admin-username=admin
admin-password=secure_password
theme=dark
language=ru
```

## 🐛 Устранение неполадок

### Частые проблемы
1. **Сервер не запускается**
   - Проверьте порты (19132 для Minecraft, 8080 для веб-панели)
   - Убедитесь в достаточности RAM
   - Проверьте логи в папке `logs/`

2. **Игроки не могут подключиться**
   - Проверьте настройки сети
   - Убедитесь в правильности IP адреса
   - Проверьте файрвол

3. **Веб-панель недоступна**
   - Проверьте, что Flask приложение запущено
   - Проверьте порт 8080
   - Проверьте логи веб-панели

## 🔄 Управление сервером

### Основные команды
```bash
./scripts/start.sh      # Запуск сервера
./scripts/stop.sh       # Остановка сервера
./scripts/restart.sh    # Перезапуск сервера
./scripts/status.sh     # Проверка статуса
./scripts/update.sh     # Обновление сервера
```

### Systemd сервис (Linux)
```bash
# Создание сервиса
sudo systemctl enable minecraft-pe-server

# Управление сервисом
sudo systemctl start minecraft-pe-server
sudo systemctl stop minecraft-pe-server
sudo systemctl status minecraft-pe-server
```
