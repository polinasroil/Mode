# 🎮 Minecraft PE Server - Мобильный Сервер

Полноценный сервер для Minecraft Pocket Edition с веб-панелью управления, написанный на Python.

## ✨ Ключевые возможности

- 🚀 **Асинхронная архитектура** на Python 3.8+
- 🌐 **Веб-панель управления** с современным интерфейсом
- 👥 **Управление игроками** (подключение, отключение, баны, кики)
- 🗺️ **Система миров** с настройками и сохранением
- 🔌 **Плагинная система** для расширения функциональности
- 📊 **Мониторинг в реальном времени** (CPU, RAM, диск)
- 🔒 **Система безопасности** с аутентификацией
- 📱 **Поддержка мобильных устройств** (Minecraft PE)
- 🎯 **REST API** для интеграции
- 📝 **Подробное логирование** всех операций

## 🛠️ Технические требования

- **Python**: 3.8+
- **ОС**: Linux, Windows, macOS
- **Память**: Минимум 512MB RAM
- **Диск**: 100MB свободного места
- **Сеть**: UDP порт 19132 (Minecraft PE), TCP порт 8080 (веб-панель)

## 🌐 Сетевой протокол

### Поддерживаемые пакеты
- `0x01` - **PACKET_LOGIN** - Вход игрока
- `0x02` - **PACKET_PLAY_STATUS** - Статус игры
- `0x05` - **PACKET_DISCONNECT** - Отключение
- `0x09` - **PACKET_TEXT** - Текстовые сообщения
- `0x0B` - **PACKET_START_GAME** - Начало игры
- `0x13` - **PACKET_MOVE_PLAYER** - Движение игрока

### Команды сервера
- `/help` - Список доступных команд
- `/time` - Текущее время в мире
- `/weather` - Текущая погода
- `/tp <x> <y> <z>` - Телепортация

## 🐛 Устранение неполадок

### Сервер не запускается
1. Проверьте версию Python (должна быть 3.8+)
2. Убедитесь, что порт 19132 свободен
3. Проверьте права доступа к файлам

### Игроки не могут подключиться
1. Проверьте настройки файрвола
2. Убедитесь, что сервер запущен
3. Проверьте логи сервера

### Веб-панель недоступна
1. Проверьте порт 8080
2. Убедитесь, что Flask запущен
3. Проверьте настройки аутентификации
