## 🚀 **БЫСТРЫЙ СТАРТ:**

### **1. Запуск сервера:**
```bash
# Переход в директорию сервера
cd minecraft-pe-server

# Запуск исправленной версии
python3 start_server_fixed.py
```

### **2. Тестирование всех протоколов:**
```bash
# В отдельном терминале
python3 test_all_protocols.py
```

### **3. Подключение с телефона:**
- 📱 Откройте **Minecraft PE**
- 🌐 Нажмите **"Играть"** → **"Серверы"**
- ➕ Нажмите **"Добавить сервер"**
- 📝 Введите:
  - **Имя сервера:** `Minecraft PE Server v2.0`
  - **IP-адрес:** `ВАШ_IP_АДРЕС`
  - **Порт:** `19132`
- ✅ Нажмите **"Добавить"** и **"Присоединиться"**

## 🔌 **ПРОТОКОЛЫ:**

### **1. RakNet протокол:**
- ✅ **Server Discovery** (ping/pong)
- ✅ **3-way handshake** для соединения
- ✅ **MTU negotiation** для оптимизации
- ✅ **ACK/NACK** для надежной доставки
- ✅ **Session management** с таймаутами

### **2. Bedrock протокол v2.0:**
- ✅ **Login sequence** для игроков
- ✅ **World initialization** с чанками
- ✅ **Player spawning** в мире
- ✅ **Movement handling** (позиция, поворот)
- ✅ **Inventory management** (предметы, броня)
- ✅ **Game events** (здоровье, опыт, способности)

### **3. Система сжатия:**
- ✅ **ZLIB compression** (основное)
- ✅ **Snappy compression** (если доступно)
- ✅ **LZ4 compression** (если доступно)
- ✅ **Адаптивное сжатие** на основе статистики
- ✅ **Заголовки сжатия** для пакетов

### **4. Система чанков:**
- ✅ **16x16 блоков** (стандарт Minecraft)
- ✅ **256 высота** мира
- ✅ **Подчанки 16x16x16** для оптимизации
- ✅ **256 типов блоков** (все базовые)
- ✅ **Генерация ландшафта** с шумом
- ✅ **Система освещения** (блоковое и небесное)
- ✅ **Биомы** (Plains, Desert, Ocean)

## 🐛 **УСТРАНЕНИЕ ПРОБЛЕМ:**

### **Проблема: "Сервер не запускается"**
```bash
# Проверьте зависимости
python3 -c "import server.raknet_protocol, server.bedrock_protocol_v2"

# Проверьте права доступа
chmod +x start_server_fixed.py

# Проверьте логи
tail -f server.log
```

### **Проблема: "Не могу подключиться с телефона"**
1. **Проверьте IP адрес** компьютера
2. **Убедитесь, что порт 19132 открыт** в файрволе
3. **Проверьте, что сервер запущен** и слушает порт
4. **Попробуйте подключиться** по локальному IP (192.168.x.x)

### **Проблема: "Игрок не загружается в мир"**
1. **Проверьте логи сервера** на ошибки
2. **Убедитесь, что чанки генерируются**
3. **Проверьте версию Minecraft PE** (должна быть 1.20.50)
