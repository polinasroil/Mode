# ⚡ Быстрый старт - Telegram Бот "Орёл или решка"

Минимальное руководство для быстрого запуска бота.

## 🚀 За 5 минут

### 1. Установка зависимостей
```bash
# Создайте виртуальное окружение
python3 -m venv venv
source venv/bin/activate  # Linux/macOS
# или
venv\Scripts\activate     # Windows

# Установите зависимости
pip install -r requirements.txt
```

### 2. Настройка бота
```bash
# Скопируйте файл конфигурации
cp .env.example .env

# Отредактируйте .env и вставьте ваш токен
nano .env  # или любой текстовый редактор
```

### 3. Получение токена
1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте токен в файл `.env`

### 4. Запуск бота
```bash
# Простой запуск
python3 bot.py

# Или используйте startup script
python3 start_bot.py
```

## 🧪 Тестирование

## 🎮 Использование

1. Найдите вашего бота в Telegram
2. Отправьте `/start`
3. Выберите режим игры:
   - 🎯 **Игра с ботом** - классическая игра
   - ⚔️ **Дуэль** - сразитесь с другом

## 📋 Команды бота

- `/start` - начать игру
- `/help` - правила и помощь
- `/stats` - ваша статистика
- `/balance` - ваш баланс

## 🛠️ Устранение проблем

### "Token is invalid"
- Проверьте правильность токена в `.env`
- Убедитесь, что токен не содержит лишних символов

### "Module not found"
- Активируйте виртуальное окружение
- Переустановите зависимости: `pip install -r requirements.txt`

### "Permission denied"
- Проверьте права доступа к файлам
- Убедитесь, что у процесса есть права на запись
