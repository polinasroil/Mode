## 🔄 Что нужно сделать для запуска

### 1. Настройка токена бота
```bash
# Скопируйте файл конфигурации
cp .env.example .env

# Отредактируйте .env и вставьте ваш токен
nano .env
```

**Содержимое .env:**
```env
BOT_TOKEN=your_actual_bot_token_here
```

### 2. Получение токена
1. Найдите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям
4. Скопируйте полученный токен в `.env`

### 3. Запуск бота
```bash
# Активируйте виртуальное окружение
source venv/bin/activate

# Запустите бота
python3 start_bot.py
```

## 🎮 Как использовать

### Для игроков
1. Найдите бота в Telegram
2. Отправьте `/start`
3. Выберите режим игры:
   - 🎯 **Игра с ботом** - классическая игра
   - ⚔️ **Дуэль** - сразитесь с другом

### Команды
- `/start` - начать игру
- `/help` - правила и помощь
- `/stats` - статистика
- `/balance` - баланс

**Решение:** Активируйте виртуальное окружение и установите зависимости
