# 🆕 Новые возможности версии 1.1.0

## 🎯 Обзор обновления

Версия 1.1.0 значительно улучшает мод "Буровая установка", добавляя глубокое бурение, умное складывание ресурсов и подробную систему уведомлений.

## 🔍 Глубокое бурение

### Что нового
- **Глубина бурения:** Увеличена с неограниченной до 100-150 блоков
- **Уведомления о глубине:** Сообщения при достижении 100+ блоков
- **Автоматическая остановка:** При достижении максимальной глубины 150 блоков

### Преимущества
- **Больше руд:** Deepslate руды появляются на глубине
- **Редкие ресурсы:** Ancient Debris, алмазы, изумруды
- **Эффективность:** Автоматическое достижение лучших слоев
- **Контроль:** Известная максимальная глубина

### Как работает
1. Установка начинает бурение с уровня размещения
2. При достижении 100 блоков выводится уведомление
3. Бурение продолжается до 150 блоков
4. Автоматическая остановка с сообщением

## 📦 Умное складывание

### Что нового
- **Автоматический поиск сундука:** В радиусе 10 блоков
- **Приоритет ближайшего:** Выбор оптимального сундука
- **Резервная система:** Выбрасывание при отсутствии сундука
- **Уведомления:** О месте складывания ресурсов

### Логика работы
1. **Поиск:** Сканирование области 20x20x20 блоков
2. **Выбор:** Определение ближайшего сундука
3. **Складывание:** Попытка поместить ресурсы
4. **Резерв:** Выбрасывание при неудаче

### Требования
- **Радиус:** Сундук в пределах 10 блоков
- **Доступность:** Сундук должен быть доступен
- **Место:** В сундуке должно быть место

## 💬 Информативность

### Сообщения в чат
- **Установка:** "Буровая установка установлена!"
- **Начало работы:** "Буровая установка начала работу!"
- **Глубина:** "Глубина бурения: X блоков"
- **Добыча:** "Добыто: [ресурсы] → Сложено в сундук"
- **Остановка:** "Достигнута максимальная глубина (150 блоков)!"

### Уведомления о работе
- **Статус:** "Буровая установка активна и работает!"
- **Прогресс:** Регулярные сообщения о глубине
- **Результаты:** Информация о добытых ресурсах
- **Ошибки:** Предупреждения о проблемах

## 🎮 Команды управления

### Проверка статуса
```
!drill status
!дрилл статус
```

**Что показывает:**
- Количество активных установок
- Примерную глубину каждой установки
- Общий статус работы

### Остановка всех установок
```
!drill stop all
!дрилл стоп все
```

**Что делает:**
- Останавливает все активные установки
- Освобождает ресурсы
- Показывает количество остановленных

## 🚀 Как использовать новые возможности

### 1. Глубокое бурение
- Устанавливайте на высоких уровнях (Y=100+)
- Следите за сообщениями о глубине
- Установка автоматически остановится на 150 блоках

### 2. Умное складывание
- Размещайте сундуки рядом с установками
- Убедитесь, что в сундуках есть место
- Следите за сообщениями о складывании

### 3. Команды управления
- Используйте `!drill status` для мониторинга
- Используйте `!drill stop all` для остановки
- Следите за сообщениями в чате
