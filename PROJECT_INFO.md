# Информация о проекте "Буровая установка"

## Обзор проекта
Мод "Буровая установка" для Minecraft PE - это комплексное решение, которое добавляет в игру автоматическую систему добычи ресурсов. Мод разработан с использованием современных стандартов Minecraft Bedrock Edition и включает в себя все необходимые компоненты для полноценной работы.

## Структура проекта

### Основные файлы
- `manifest.json` - главный файл манифеста мода
- `behavior_pack.json` - конфигурация поведенческого пакета
- `resource_pack.json` - конфигурация ресурсного пакета
- `world_settings.json` - настройки мира для мода

### Скрипты
- `scripts/main.js` - основной JavaScript код мода

### Блоки и предметы
- `blocks/drill_rig.json` - поведение блока буровой установки
- `items/drill_rig.json` - настройки предмета
- `models/blocks/drill_rig.geo.json` - геометрия блока
- `animations/drill_rig.animation.json` - анимации

### Ресурсы
- `textures/` - текстуры блоков и предметов
- `texts/` - файлы локализации (русский и английский)

### Игровая механика
- `recipes/drill_rig.json` - рецепт крафта
- `loot_tables/blocks/drill_rig.json` - таблица добычи
- `trading/drill_rig.json` - торговля с жителями

### Документация
- `README.md` - основная документация
- `INSTALLATION.md` - инструкции по установке
- `PACKAGING_INSTRUCTIONS.md` - инструкции по упаковке
- `LICENSE` - лицензия MIT
- `VERSION` - информация о версии
