# 📋 Итоговый отчет - Telegram Бот "Орёл или решка"

## 🎯 Обзор проекта

Полнофункциональный игровой бот для Telegram с игрой "Орёл или решка", включающий режимы игры с ботом и дуэли между игроками. Проект разработан с использованием современных технологий и лучших практик.

## 🎮 Игровые возможности

### Режимы игры
1. **Игра с ботом**
   - Выбор ставки (от 1 до 1000)
   - Выбор стороны монеты (Орёл/Решка)
   - Мгновенный результат
   - Выигрыш = двойная ставка

2. **Дуэли между игроками**
   - Создание дуэли с уникальной ссылкой
   - Оба игрока делают ставки
   - Оба выбирают сторону монеты
   - Победитель забирает все ставки

### Команды бота
- `/start` - начать игру
- `/help` - правила и помощь
- `/stats` - статистика игр
- `/balance` - текущий баланс

## 🚀 Установка и запуск

### Быстрый старт
```bash
# 1. Клонирование
git clone <repository-url>
cd telegram-coin-flip-bot

# 2. Установка зависимостей
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# 3. Настройка
cp .env.example .env
# Отредактируйте .env и вставьте ваш BOT_TOKEN

# 4. Запуск
python3 start_bot.py
```

### Тестирование
```bash
# Проверка всех компонентов
python3 test_import.py

# Результат должен быть:
✅ All tests passed! The bot is ready to run.
```
